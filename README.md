## ระบบนี้คืออะไร

ระบบวิเคราะห์ข่าวกรองยานพาหนะต้องสงสัยบนสายทาง
อ่านข้อมูล LPR จากจุดตรวจ -> วิเคราะห์อัตโนมัติ -> แสดงรายชื่อเป้าหมายเร่งด่วน

Admin อัปโหลด CSV -> ระบบวิเคราะห์ -> ผู้บังคับบัญชาดูผลผ่านเว็บ (ทุกอุปกรณ์)

---

## ระดับสิทธิ์ผู้ใช้

| Role           | สิทธิ์                        |
|----------------|-------------------------------|
| Super Admin    | ทุกอย่าง + จัดการ User        |
| Admin          | อัปโหลดข้อมูล + ดูผล          |
| Viewer         | ดูผลวิเคราะห์ + Export เท่านั้น |

## ขั้นตอนติดตั้ง (Admin เครื่องใหม่)

### สิ่งที่ต้องมีก่อน
- Windows 10/11
- Python 3.10 ขึ้นไป -> ดาวน์โหลดที่ https://python.org
  !! ติ๊ก "Add Python to PATH" ตอนติดตั้ง

### ขั้นตอน

1. รับโฟลเดอร์โปรแกรม จาก Super Admin

2. แก้ไขไฟล์ .streamlit\secrets.toml
   เปิดด้วย Notepad ใส่ข้อมูลที่ Super Admin ให้:
   [supabase]
   url = "https://xxxxx.supabase.co"
   key = "eyJxxxxxxxxxxxx"

3. ดับเบิลคลิก Start_Program.bat
   ระบบติดตั้ง dependencies อัตโนมัติ (ครั้งแรก ~3-5 นาที)
   Browser จะเปิดที่ http://localhost:8501 อัตโนมัติ

4. Login ด้วย Username/Password ที่ได้รับมา

---

## การใช้งานประจำวัน (Admin)

### เปิดระบบ
ดับเบิลคลิก Start_Program.bat -> รอ Browser เปิด -> Login

### อัปโหลดข้อมูล
1. เลือกเมนู Admin Portal
2. Tab นำเข้าข้อมูล
3. เลือกไฟล์ CSV จากจุดตรวจ (รับได้ทีละหลายไฟล์)
4. กด "ตรวจสอบคุณภาพข้อมูล" -> รอผล
5. กด "ยืนยันและประมวลผล" -> รอ ~1-3 นาที
6. ระบบ Sync ขึ้น Cloud อัตโนมัติ

หากมีหลาย Admin อัปโหลดวันเดียวกัน ระบบ Merge ข้อมูลรวมให้อัตโนมัติ

### ปิดระบบ
กด Ctrl+C ในหน้าต่าง Command Prompt หรือปิดหน้าต่างนั้นได้เลย

## ลืมรหัสผ่าน / เปลี่ยนรหัสผ่าน

Super Admin:
1. Login ด้วย supuseradmin
2. Admin Portal -> Tab จัดการผู้ใช้
3. เลือก User -> กรอกรหัสผ่านใหม่ -> บันทึก

Admin/Viewer: ติดต่อ Super Admin เพื่อเปลี่ยนให้

## ปัญหาที่พบบ่อย

| ปัญหา                    | วิธีแก้                                        |
|--------------------------|------------------------------------------------|
| Browser ไม่เปิด          | เปิด Chrome เอง -> http://localhost:8501       |
| Login ไม่ได้             | ตรวจ Caps Lock / ติดต่อ Super Admin            |
| อัปโหลดแล้วไม่เห็นผล    | กด รีเฟรชข้อมูล ใน Sidebar                    |
| Sync Error               | ตรวจอินเทอร์เน็ต แล้วอัปโหลดใหม่              |
| โปรแกรมช้ามาก            | ปิดโปรแกรมอื่น / ลองแบ่งอัปโหลด               |
| หน้าจอขาว / Error แดง    | กด F5 หรือปิด-เปิด Browser ใหม่               |

## โครงสร้างไฟล์สำคัญ

itrap_agent/
  Start_Program.bat    <- ดับเบิลคลิกเปิดระบบ
  app.py               <- โปรแกรมหลัก (ห้ามลบ)
  cloud_app.py         <- หน้า Cloud
  auth.py              <- ระบบ Login
  supabase_sync.py     <- ระบบ Cloud Sync
  requirements.txt     <- รายการ Library
  .streamlit/
    secrets.toml       <- !! ข้อมูลลับ (ห้ามส่งต่อ)
  data/
    memory.parquet     <- ฐานข้อมูลท้องถิ่น

!! ห้ามลบไฟล์ใดๆ ในโฟลเดอร์นี้
!! ห้ามส่ง secrets.toml ให้ผู้อื่นทาง Line/Email
